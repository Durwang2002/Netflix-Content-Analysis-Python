# Netflix Content Analysis Using Python

## Project Overview

This project analyzes Netflix's global content library to uncover trends in content distribution, growth, genre popularity, country contribution, audience targeting, and business strategy.

## Business Problem

Netflix hosts thousands of movies and TV shows worldwide. Understanding content distribution and growth patterns can help improve content acquisition and investment strategies.

## Objectives

* Analyze content type distribution
* Identify top content-producing countries
* Understand genre popularity
* Analyze content growth trends
* Evaluate audience targeting through ratings
* Generate business recommendations

## Key Insights

* Movies dominate Netflix content library.
* Content production is concentrated in a few countries.
* Netflix experienced rapid content expansion in recent years.
* Certain genres consistently outperform others.
* Ratings reveal strong focus on mature audiences.

## Recommendations

* Expand content acquisition in emerging regions.
* Increase genre diversification.
* Strengthen partnerships with successful creators.
* Balance investments between movies and TV shows.
